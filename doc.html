About

KatLogger is a Python-based keylogger developed Kat2800 and Lif28. It records all keystrokes into a log.txt file, which is created upon the first execution of KatLogger.
The initial release, "KatLogger_1.0.zip", is compatible only with Windows. A new version is in development to support both Linux and Windows platforms, with plans to add features
such as establishing backdoors to connect victim computers to ours, and more.

How Does it Work?

KatLogger is written in Python and it records all keystrokes into the log.txt file.
The source code is open-source, with the main script located in the python folder as KatLogger.py.
Upon first execution, KatLogger creates a startup entry to ensure it runs automatically when the system boots.
It also hides all files on the USB drive. Therefore, if the computer is shut down and restarted, KatLogger will continue to operate as intended.
